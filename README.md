Chess64 3D(3dchess.app) ♟️

Chess64 3D is a premium, browser-based chess application that merges professional-grade engine analysis with a sleek, high-fidelity 3D aesthetic. Built using Three.js for rendering and Stockfish.js for intelligence, it provides a seamless experience for players, analysts, and chess enthusiasts.

🌟 Features

🎮 Gameplay Modes

Free Analysis: Explore lines and evaluate positions without restrictions.

Vs Computer: Test your skills against Stockfish (Levels 1-20, ~400 to 3300 ELO).

Computer vs Computer: Watch the engine play against itself with adjustable styles (Aggressive, Solid, Positional, Gambiteer).

🎨 Customization & Aesthetics

Material System: Choose between Matte, Metallic, Glass, Neon Glow, Luxury Gold, and Carbon Fiber piece sets.

Thematic Boards: obsidian, Walnut, Emerald, and Cyberpunk themes.

Visual Effects: Dynamic glare colors (Electric Blue, Ruby Red, Solar Gold) and optional background typography.

Toggleable View: Switch instantly between an immersive 3D Perspective and a clean Modern 2D layout.

🧠 Analysis Tools

Live Eval Bar: Real-time evaluation of the current position.

Standalone Best Move HUD: A dedicated indicator showing the engine's top suggestion, visible even when the main UI is hidden.

Algebraic Move List: Professional move history with standard notation.

PGN Support: Quickly copy your game's PGN to the clipboard for external review.

📱 Responsive & Intuitive

Full Touch Support: Intuitive pinch-to-zoom and rotation gestures for mobile and tablet users.

Responsive Design: Adapts flawlessly to any screen size or orientation.

Haptic Audio: Crisp, low-latency move sounds.

🚀 Getting Started

Prerequisites

No installation is required. The project runs entirely in the browser using modern web technologies (HTML5, CSS3, JavaScript).

Deployment

To host this yourself, simply upload index.html to any static hosting provider (GitHub Pages, Vercel, Netlify).

🛠️ Built With

Three.js - 3D Graphics Rendering

Stockfish.js - Powerful Chess Engine (WebAssembly/Worker)

Chess.js - Chess Logic and Move Validation

Tailwind CSS - Modern UI Styling

📖 How to Use

Move Pieces: Click/Tap a piece to select it, then click/tap the destination square.

Camera Controls:

Rotate: Drag with one finger or mouse.

Zoom: Pinch with two fingers or use the scroll wheel.

Recenter: Use the "Recenter" button in the bottom navigation.

Hide UI: Toggle the UI with the "Hide UI" button for a focused, cinematic view.
